# Extension Marketplace

This package is intentionally retired for Smarty Code Milestone 1.

The desktop app uses the bundled extension registry only. Runtime loading
strips hosted download URLs and screenshots from bundled entries before they
reach the renderer, and final proof must show no marketplace egress.

Do not add Worker, bucket, namespace, route, Wrangler, publish, or deploy
configuration here unless a later milestone retargets marketplace hosting to a
verified Smarty-owned resource with the same resource-audit and redaction gates
as sync/auth.
